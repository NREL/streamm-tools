.. _examples:

Examples
********

To run the examples navigate to the examples directory in the streamm release, and run the Jupyter Notebook::

    $ jupyter notebook

This will open the notebook Home page in your default browser.
For more information on Jupyter Notebooks check their site `<http://jupyter.org/>`_.
Then, click on the notebook to get a save instance of the notebook.
You can restart the notebook by clicking Kernal>Restart & Clear Output, then hitting 'Control'+'Shift' to execute each cell.

The Jupyter notebooks structures.ipynb, forcefields.ipynb, buildingblocks.ipynb and calculations.ipynb have been provided you examples of the core functions of each module.  To complete the examples you will need to be able to view molecular geometeries using a viewer such as  `VMD <http://www.ks.uiuc.edu/Research/vmd/>`_ or `Avogadro <https://avogadro.cc/>`_. To run examples for each software package LAMMPS, NWChem and GAUSSIAN, you will need to have them installed on your local machine or have access to resource that has them installed.
