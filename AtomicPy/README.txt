Separate directory for Travis python script development


===========

Is a set of tools to create and modify structure files for atomic and molecular simulations. 


----------------------------------DIRECTORIES ----------------------------------------------

src/

Contians python code for reading and write files from various packages:

GAUSSIAN
NWCHEM

GROMACS
LAMMPS

As well as general structure manipulations

et/

Contains scripts which can calculate the electron coupling matrix element based on an MD snapshot. 

md/

Contains scripts for modifying and analyzing output of an molecular dynamics simulation

torsion/

Contains scripts calculating  torsional potentials for use in classical force-fields
